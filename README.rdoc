== Regional Business API

This early implementation of the Regional Buiness Api is designed to fetch Web API servers for Buiness related data. Given bounding regions to collect the data. The data would be accessible by a RESTful api. 

This project is intended by experimental and research purpose. There is no intention for real usage on the code to collect and persist business data from Web API source.

This Readme file would gradually be updated as the Collection and RESTful API functionalities being developed.
